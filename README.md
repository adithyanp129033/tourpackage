# Tour Package Booking Web Application

A full-stack tourism and adventure camp booking platform built with React, Django, and Django REST Framework. The application allows users to explore, compare, and book tour packages and adventure camp experiences through a modern and responsive interface.

## Features

- User registration, login, and profile management
- Secure authentication using JWT/Session Authentication
- Browse and search tour packages and adventure camps
- Filter packages by location, budget, duration, ratings, and activities
- View detailed package and camp information
- Online booking and booking management
- RESTful API integration using Django REST Framework
- Responsive frontend built with React
- Admin panel for managing packages, camps, users, and bookings
- Secure database management and user data protection

## Tech Stack

### Frontend
- React.js
- HTML5
- CSS3
- Bootstrap

### Backend
- Django
- Django REST Framework (DRF)

### Database
- SQLite / MySQL

### Authentication
- JWT Authentication / Django Authentication System

## Project Objective

The goal of this project is to provide a centralized digital platform that simplifies tourism and adventure camp bookings by offering an efficient, secure, and user-friendly experience for both travelers and administrators.
